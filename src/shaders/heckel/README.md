# Heckel shader references

GLSL fragments extracted from [Maxime Heckel's R3F playground](https://r3f.maximeheckel.com/)
(`lego`, `sdf3`–`sdf13`) for study and adaptation. MIT-licensed upstream
([blog.maximeheckel.com](https://github.com/MaximeHeckel/blog.maximeheckel.com)).

The live hero matrix rain (`src/MatrixRain.jsx`) uses WebGPU and borrows pointer
colour-shift ideas from `sdf3.frag.glsl`.
